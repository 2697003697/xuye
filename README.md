# 续页

一个浏览器扩展，自动加载搜索引擎后续页面，合并多页结果。

## 功能特点

- 🔄 **自动加载**：滚动到页面底部时自动加载下一页
- 📄 **内容合并**：将多页搜索结果无缝拼接在一起
- 🔍 **多引擎支持**：支持 Google、百度、Bing 等主流搜索引擎
- 📊 **实时统计**：显示已加载页数和结果数量
- ⚙️ **可配置**：支持设置最大加载页数

## 安装方法

### Chrome / Edge 浏览器

1. 下载并解压扩展文件
2. 打开浏览器，进入扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择扩展文件夹

## 使用方法

1. 安装扩展后，访问支持的搜索引擎（Google、百度、Bing）
2. 进行搜索后，向下滚动页面
3. 扩展会自动加载并拼接下一页的内容
4. 页面右上角会显示已加载的页数和结果数量

## 配置选项

点击浏览器工具栏上的扩展图标，可以：
- 启用/禁用自动加载功能
- 设置最大加载页数（2-10页）

## 注意事项

1. 扩展需要访问搜索引擎页面的权限
2. 某些搜索引擎可能有反爬虫机制，过度使用可能导致暂时无法加载
3. 建议根据网络情况适当调整最大加载页数

## 隐私说明

- 扩展仅在搜索引擎页面运行
- 不会收集或上传任何用户数据
- 所有操作都在本地完成
